# Number

A component for rendering numbers.

## Usage

```jsx live
function App(props) {
  return (
    <LaunchedWrapper>
      <Number tag="examplenumber">3</Number>
    </LaunchedWrapper>
  );
}
```

## Props

| Name     | Type                                                                                                                                                           | Default | Description                                                                           |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- | ------------------------------------------------------------------------------------- |
| tag      | `string`                                                                                                                                                       | `null`  | The number's associated tag. See [tags](/Get%20Started/concepts#tagging) for details. |
| element  | [`HTMLTextTagsType`](https://github.com/launchsite-tech/launched/blob/5013cfc2a6dd1e7f1e9d84a8c612ff1997859fcb/src/ui/components/helpers/elementGroups.ts#L36) | `"p"`   | The HTML element to render the number under.                                          |
| children | [`ReactNode`](https://react.dev/learn/typescript#typing-children)                                                                                              | `null`  | The number to render.                                                                 |
