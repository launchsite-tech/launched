# Text

A component for rendering plain text.

## Usage

```jsx live
function App(props) {
  return (
    <LaunchedWrapper>
      <Text tag="exampletext">Hello, world!</Text>
    </LaunchedWrapper>
  );
}
```

## Props

| Name     | Type                                                                     | Default | Description                                                   |
| -------- | ------------------------------------------------------------------------ | ------- | ------------------------------------------------------------- |
| tag      | `string`                                                                 | `null`  | The text's associated [tag](/Get%20Started/concepts#tagging). |
| element  | `"p" \| "h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "span" \| "div"` | `"p"`   | The HTML element to render the text under.                    |
| children | [`ReactNode`](https://react.dev/learn/typescript#typing-children)        | `null`  | The text to render.                                           |
