## Usage

<Tabs groupId="mode">
<TabItem label="React" value="react">

Wrap your app with the `LaunchedProvider` to enable Launched component rendering:

```jsx title="src/index.jsx"
/* Import the Launched provider */
import { LaunchedProvider } from "launched";

...

/* Add the provider to your app */
root.render(
  <LaunchedProvider>
    <App />
  </LaunchedProvider>
);
```

Import the premade Launched components to start managing your content:

```jsx title="src/app.jsx"
/* Import the Launched Text element */
import { Text } from "launched/components";

...

export default function App() {
  return (
    <div>
      <Text tag="title" element="h1">Hello, world!</Text>
      <Text tag="description">Welcome to my site.</Text>
    </div>
  )
}
```

</TabItem>
<TabItem label="Static HTML" value="static">

Initialize Launched:

```html title="src/index.html"
<script type="module" defer>
  const { Launched } = launched;

  new Launched({
    mode: "static", // You must include this line!
    save: (data) => console.log(data),
  });
</script>
```

Add Launched attributes to the elements you want to be editable:

```html title="src/index.html"
<body>
  <h1 data-tag="title">Hello, world!</h1>
  <p data-tag="description">Welcome to my site.</p>
</body>
```

</TabItem>
</Tabs>

And that's all! You're ready to start using Launched. To preview edit mode, add the `?mode=edit` query parameter to your URL.
